## ğŸ¯ Project Overview
This project involves creating a professional website for a small plumbing business. The website targets small business owners in Pretoria and Johannesburg, South Africa, providing comprehensive information about plumbing services, contact details, and an enquiry system. 

The site is built using modern web technologies (HTML5, CSS3, and JavaScript) with a focus on:
- User-friendly navigation
- Responsive design that works on all devices
- Professional aesthetic with modern gradients and animations
- Clear call-to-action elements
